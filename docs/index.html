01 · Capabilities

Purpose-built for the
Resolve workflow

Every feature solves a real pain point in the DaVinci Resolve Free on Linux pipeline. Nothing more, nothing less.

01

mpv-powered preview

Embedded mpv video output renders directly inside the GUI via X11 child window. Full playback control with frame-accurate stepping — jump, scrub, or nudge one frame at a time.

02

Keyboard-first marking

Set in/out points with I and O while playing. Each O auto-captures a segment. Build a batch of clips without ever touching the mouse.

03

DNxHD / DNxHR export

Transcode only the segments you need. ffmpeg encodes to DNxHD or DNxHR in LB, SQ, HQ, or HQX profiles. Output is a .mov file DaVinci Resolve reads natively.

04

Segment management

Capture multiple in/out ranges from the same source file. Review each segment's timing and duration in the sidebar. Remove any before export.

05

Drag & drop

Open files by dragging them onto the preview area, or use Ctrl+O for the native file dialog. Supports mp4, mov, mkv, avi, webm, mts, and more.

06

Smart export naming

Output files are automatically named with source filename and segment timestamps. Prompts before overwriting anything that already exists.

02 · Architecture

Three processes,
one seamless pipeline

dnclip orchestrates three battle-tested tools — each doing what it does best — through a unified Rust GUI.

GUI Layer
egui + eframe
→ ↓
Playback Engine
mpv
→ ↓
Encoder
ffmpeg

egui renders the UI

Immediate-mode GUI in Rust. Timeline slider, transport controls, segment list, and export panel — all rendered at native speed with minimal resource use.

mpv decodes & renders video

Spawned as a child process, embedded into the egui window via X11 reparenting. Controlled over a Unix socket with JSON IPC — play, pause, seek, frame-step, query properties.

ffmpeg encodes the output

Invoked per export with precise -ss and -t parameters. Encodes to DNxHD/DNxHR with PCM audio in a .mov container — Resolve-ready.

03 · Technical

Tech specs

Language Rust (edition 2021) — 1,411 lines across 6 source files
GUI framework egui 0.34 with eframe runtime
Video playback mpv, embedded via X11 child window + JSON IPC
Export encoder ffmpeg CLI — DNxHD/DNxHR profiles LB, SQ, HQ, HQX
Output format QuickTime .mov — DNx video + PCM 16-bit audio
Windowing X11 (x11-dl) for native child window embedding
Serialization serde + serde_json for mpv IPC protocol
File dialogs rfd (native) + egui-file-dialog (in-GUI)
Packaging cargo-deb with .desktop entry and app icon
Input formats mp4 mov mkv avi webm mts m2ts ts flv wmv

04 · Getting started

Install

Dependencies
# Ubuntu / Debian sudo apt install mpv ffmpeg libx11-dev
Install with Cargo
$ cargo install dnclip
Or build from source
$ git clone https://github.com/yinguobing/dnclip.git $ cd dnclip $ cargo build --release $ ./target/release/dnclip
